# 노벨피아 EP 필터

노벨피아 랭킹 페이지에서 **최소 편수와 완결 여부를 기준으로 작품을 필터링**하는 간단한 브라우저 확장 / 사용자 스크립트입니다.

조건에 맞지 않는 작품을 **화면에서 숨겨** 랭킹을 더 편하게 탐색할 수 있도록 도와줍니다.

이 프로젝트는 원래 개인 사용을 위해 제작되었으며 **Unlicense(퍼블릭 도메인)**로 공개되었습니다.

---

# 기능

* **최소 편수 기준 필터**
* **완결 작품 제외 옵션**
* 필터링된 작품 개수 표시
* 노벨피아 랭킹 페이지에서 자동 동작
* 외부 서버 요청 없음
* 사용자 데이터 수집 없음

---

# 설치 방법 (Chrome 확장 프로그램)

1. 이 저장소를 **ZIP으로 다운로드**
2. ZIP 파일 압축 해제
3. Chrome에서 아래 주소로 이동

```
chrome://extensions
```

4. **개발자 모드 활성화**
5. **압축해제된 확장 프로그램 로드** 클릭
6. `manifest.json`이 포함된 폴더 선택

---

# Tampermonkey 사용자 스크립트 사용 방법

이 프로젝트는 **Tampermonkey 사용자 스크립트로도 사용할 수 있습니다.**

### 1. Tampermonkey 설치

Chrome 웹스토어에서 **Tampermonkey** 확장 프로그램을 설치합니다.

### 2. 스크립트 설치

1. Tampermonkey 대시보드 열기
2. **새 스크립트 생성**
3. 저장소에 포함된 **userscript 코드**를 붙여넣기
4. 저장

---

# Tampermonkey 권한 설정

Tampermonkey로 사용할 경우 Chrome 확장 설정에서 다음 권한을 활성화해야 합니다.

1. Chrome에서 아래 주소로 이동

```
chrome://extensions
```

2. **Tampermonkey 확장 프로그램 찾기**
3. **세부정보 (Details)** 클릭
4. 다음 옵션 활성화

* **파일 URL에 대한 액세스 허용 (Allow access to file URLs)**
* **사이트 데이터 읽기 및 변경 권한 허용**

권한이 비활성화된 경우 스크립트가 정상적으로 동작하지 않을 수 있습니다.
