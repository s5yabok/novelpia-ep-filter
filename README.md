# Novelpia EP Filter

A simple Chrome extension that filters works on Novelpia ranking pages based on minimum episode count and completion status.

This extension removes works that do not meet the selected criteria directly from the page, making it easier to browse rankings.

This project was originally created for personal use and is released to the public domain under the Unlicense.


## Features

- Filter works by **minimum episode count**
- Option to **exclude completed works**
- Shows how many works were filtered
- Runs directly on Novelpia ranking pages
- No external requests
- No data collection


## Installation (Chrome)

1. Download this repository as ZIP  
2. Extract the ZIP file  
3. Open Chrome and go to:

   chrome://extensions

4. Enable **Developer mode** (top right)
5. Click **Load unpacked**
6. Select the extracted folder containing `manifest.json`


## Notes

- This extension only modifies how the page is displayed in your browser.
- It does not collect or transmit any user data.
- All settings are stored locally in the browser.


## Disclaimer

This project is **not affiliated with Novelpia**.

# 노벨피아 EP 필터

노벨피아 랭킹 페이지에서 **최소 편수와 완결 여부를 기준으로 작품을 필터링**하는 간단한 Chrome 확장 프로그램입니다.

조건에 맞지 않는 작품을 페이지에서 제거하여 랭킹을 보다 편하게 탐색할 수 있도록 도와줍니다.

이 프로젝트는 원래 개인 사용을 위해 제작되었으며, 현재는 **Unlicense(퍼블릭 도메인)**로 공개되었습니다.


## 기능

- **최소 편수 기준 필터**
- **완결 작품 제외 옵션**
- 필터링된 작품 개수 표시
- 노벨피아 랭킹 페이지에서 동작
- 외부 서버 요청 없음
- 사용자 데이터 수집 없음


## 설치 방법 (Chrome)

1. 이 저장소를 **ZIP으로 다운로드**
2. ZIP 파일 압축 해제
3. Chrome에서 다음 주소로 이동

   chrome://extensions

4. **개발자 모드 활성화**
5. **압축해제된 확장 프로그램 로드** 클릭
6. `manifest.json`이 포함된 폴더 선택


## 참고

- 이 확장은 브라우저에서 페이지 표시 방식만 변경합니다.
- 사용자 데이터를 수집하거나 전송하지 않습니다.
- 설정 값은 브라우저 로컬에 저장됩니다.
